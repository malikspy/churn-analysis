Predicting Customer Churn Using Machine Learning – Telco Customer Dataset

Project Summary

Customer churn is a major problem for telecommunications companies. In this project, I analyzed the factors influencing churn and built a machine learning model to predict customers who are likely to churn, enabling the company to implement more effective retention strategies.

Objectives

• Identify key patterns and factors causing churn.

• Predict customers who are likely to churn.

• Provide retention strategy recommendations based on customer segmentation.
